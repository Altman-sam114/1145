# v4.74 / 舰炮命中水柱与冲击圈

## 版本号与依据

v4.74。v4.73 已由最新 `origin/main` run `29122388745` artifact 验收通过，海军具备方向航迹；下一项限定为可见水面舰炮命中反馈。

## 参考与背景

Agent X 检索了 Noble Master 官方 Desert Stormfront 产品页、官方预告片入口和官方截图资源。参考画面采用俯视等距、低饱和地形和短促高对比战斗反馈。本轮用窄水柱、椭圆冲击圈、冠状泡沫和少量碎滴强化海战命中，不采用长期遮挡舰体的大面积烟雾或粒子系统。

## 目标

- Battleship、Coastal Battery、Artillery 命中水面舰艇时显示紧凑水柱和水面扩散圈。
- 玩家舰艇被命中始终可见；敌方舰艇只有玩家已知时显示反馈。
- 玩家与 AI 共享 `fire(attacker:target:)` 链路。
- Submarine 保持既有 `ASW HIT`，Carrier 保持 wing-strike，不叠加错误反馈。
- CI capture scene 保留确定性水柱，能经长 runner 延迟截图。

## 非目标

- 不修改伤害、射速、射程、护甲、目标合法性、AI、路径、经济、任务、胜负、声呐或潜艇暴露。
- 不新增外部素材、粒子系统、依赖或 XCTest target。
- 不运行本地测试、build、模拟器或探针。
- 不提交用户的 `project.pbxproj` 团队号改动。

## 实现步骤

1. 增加 `shouldShowNavalWaterImpact(...)`，限定攻击者、目标域和玩家可见性。
2. 在 `fire(...)` 伤害结算旁调用只读视觉 helper，命中点按目标 id 确定性偏离舰体中心。
3. `showNavalWaterImpact(...)` 创建短时冲击圈、水柱、泡沫冠和四个碎滴，结束后整体移除。
4. CI capture mode 使用 persistent 参数保留 BB/CV 附近两个效果，普通战斗仍自动消失。
5. 更新 README、flow、flowchart、test 和本提示词。
6. commit `v4.74: 增强舰炮水面命中反馈`，直推 `origin/main`，只做云端验证。

- 三类舰炮命中可见水面舰艇时出现清楚但不遮挡 HUD 的水柱、冲击圈和碎滴。
- 隐藏敌舰不通过效果泄露，潜艇不叠加水面水柱。
- 效果节点在普通战斗 0.6 秒内移除，不持续增长。
- 最新 main manifest SHA 一致，static checks、lint、generic iOS build、simulator launch 成功，JUnit 0 失败。
- coast screenshot 同时显示 BB/CV、航迹和至少两个水柱 / 冲击圈，非黑白屏。
