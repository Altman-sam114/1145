# v4.76 / 空军模型几何细化

## 版本号与依据

v4.76。v4.75 已由最终 `origin/main` run `29125469824` artifact 验收通过，空军已有投影、导弹尾迹和命中反馈；下一项限定为 HEL/JET 机体细节。

## 参考与背景

Agent X 再次核对 Desert Stormfront 官方截图、官方 trailer 和 15 分钟 gameplay。移动端俯视视角需要用高对比轮廓和少量关键部件表达机型，不能依赖细碎纹理。本轮全部使用现有 SpriteKit 几何，并控制在当前 footprint 内。

## 目标

- Helicopter 增加座舱、双侧武器短舱、旋翼盘 / 轴心、动画尾桨、起落架支杆和滑橇。
- Fighter 增加座舱、机身脊线、双侧翼下挂弹 / 挂架、尾部稳定翼和机鼻传感器。
- Blue / Red 继续使用阵营机身色，座舱玻璃使用蓝 / 橙差异色。
- 所有部件作为现有 `base` 子节点，自动继承移动、镜像、迷雾、选择和销毁。
- 继续使用 v4.75 air screenshot 验证两方 HEL/JET。

## 非目标

- 不新增弹药、挂载消耗、武器切换、碰撞、飞行状态或模型素材。
- 不修改速度、路径、伤害、射程、AI、Carrier guard、经济、任务、胜负或迷雾。
- 不新增依赖或 XCTest target。
- 不运行本地测试、build、模拟器或探针。
- 不提交用户 `project.pbxproj` 团队号改动。

1. 在 `addAirUnitBody(...)` 复用阵营 bodyColor，并增加阵营化 glassColor。

3. JET 在原 `jetPath()` 上叠加脊线、座舱、挂架、两枚挂弹、稳定翼和机鼻传感器。
4. 更新 README、flow、flowchart、test 和本提示词。
5. commit `v4.76: 细化空军机体模型`，直推 `origin/main`，只做云端验证。
