# v4.30（玩家潜艇已知声呐接触状态）

## 版本号

v4.30

## 版本分配依据

- `update_log.md` 当前最新正式玩法版本为 v4.29「支援按钮资金不足提示」。
- v4.22 / v4.27 已增强玩家 active sonar sensor 对已知敌方潜艇 contact count 的 HUD 表达。
- 当前玩家单选己方 Submarine 时，选择面板只显示 `Temporary detected Ns` 或 `Stealth while undetected`，没有区分“是否处于玩家已知敌方声呐覆盖内”。
- 本轮是潜艇/声呐 HUD 可读性的窄范围延伸，只做玩家单选己方潜艇的只读状态，不改变侦测规则或隐藏信息边界。

## 背景

当前 `DesertFrontline/GameScene.swift` 中：

- `EntityKind.hasSonar` 包含 Battleship、Submarine、Carrier、Helicopter 和 Sonar Buoy。
- `isActiveSonarSensor(_:)` 过滤存活实体和已完工结构。
- `isSubmarineDetected(_:by:)` 是实际潜艇可见性判断，包含 `revealedUntil` 和对应阵营 active sonar sensor 覆盖。
- `submarineStatusLine(for:)` 当前在 `revealedUntil` 有效时显示临时暴露倒计时，否则显示 `Stealth while undetected`。
- `isKnownToFaction(..., observer: .player)` 是玩家已知边界；新增 HUD 信息不能通过隐藏敌方 sensor 泄露玩家本不该知道的敌方位置或能力。

## 目标

1. 增强玩家单选己方 Submarine 的状态行。
   - 如果 `revealedUntil > lastUpdateTime`，继续显示 `Temporary detected Ns`。
   - 否则如果该玩家 Submarine 位于至少一个玩家已知、敌方 active sonar sensor 的 `sonarRange(for:)` 内，显示 `Known sonar contact`。
   - 否则显示 `Stealth / no known contact`。
2. 新增 helper 必须尊重玩家已知边界。
   - 只统计 `sensor.faction == .enemy`。
   - sensor 必须 `isActiveSonarSensor(sensor)`。
   - sensor 必须 `isKnownToFaction(sensor, observer: .player)`，避免隐藏敌方 sonar sensor 通过 HUD 泄露。
   - sensor 与玩家潜艇距离必须小于等于 `sonarRange(for: sensor.kind)`。
3. 保持真实潜艇侦测链路完全不变。
   - 不改变 `isSubmarineDetected(_:by:)`。
   - 不改变 `isKnownToFaction(...)`。
   - 不改变 `revealedUntil` 写入、支援命中暴露、开火暴露、direct-fire ASW HIT 反馈、fog 集合或实体隐藏。
4. 同步 README、flow 和必要的 flowchart 文案。

## 非目标

- 不改变任何单位、建筑、声呐范围、攻击、伤害、视野、费用、生产、支援或 AI 行为。
- 不把隐藏敌方声呐 sensor 计入玩家可见的 `Known sonar contact` 文案。
- 不新增地图标记、范围圈、按钮、图标、颜色、动画、音效或交互。
- 不改变多选 `Ctc n` 摘要、单选 sonar contact count、`sonarContactCount(for:)` 或 `groupSonarContactCount(for:)`。
- 不改变敌方潜艇状态行、敌方侦测规则、Red AI 动态 ASW、Red SCAN 决策或目标合法性。
- 不更新 `md/test/test.md`，因为测试流程不变。
- Agent B 不更新 `update_log.md`；Agent C 在 artifact 验收通过后追加正式记录。

2. 修改 `DesertFrontline/GameScene.swift`。

推荐实现方向：

```swift
private func submarineStatusLine(for entity: GameEntity) -> String {
    if entity.revealedUntil > lastUpdateTime {
        return "Temporary detected \(Int(ceil(entity.revealedUntil - lastUpdateTime)))s"
    }
    if entity.faction == .player && isCoveredByKnownEnemySonar(entity) {
        return "Known sonar contact"
    }
    if entity.faction == .player {
        return "Stealth / no known contact"
    }
    return "Stealth while undetected"
}

private func isCoveredByKnownEnemySonar(_ submarine: GameEntity) -> Bool {
    guard submarine.kind == .submarine,
          submarine.faction == .player,
          submarine.isAlive else { return false }
    return entities.values.contains { sensor in
        sensor.faction == .enemy &&
            isActiveSonarSensor(sensor) &&
            isKnownToFaction(sensor, observer: .player) &&
            sensor.node.position.distance(to: submarine.node.position) <= sonarRange(for: sensor.kind)
    }
}
```

约束：

- helper 只读，不能写 `revealedUntil`、fog 集合、entity hidden state、selection 或任何命令状态。
- `Known sonar contact` 只代表玩家已知敌方 sonar sensor 覆盖，不代表所有隐藏敌方侦测都被揭示。
- 不调用或修改 `isSubmarineDetected` 来改变真实侦测规则。
