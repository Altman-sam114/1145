# v4.94 岸防炮模型与海岸交战

## 版本号

`v4.94`

## 版本分配依据

- `update_log.md` 与 `md/prompt/v4（海军航母）/` 的最新正式版本为 `v4.93`。
- 人工总目标要求 UI、战斗、模型、地图、海战和空战持续细化；v4.93 已完成 Battleship 模型与舰炮齐射，但 Coastal Battery 仍是单炮管简模，与当前双发齐射反馈不一致。本轮只统一岸防炮模型、开火反馈和云端海岸取证，因此递增为 `v4.94`。

## 背景

- 参考对象固定为百度百科 lemma id `4042982` 对应的 Noble Master Games《沙漠风暴 Desert Stormfront》，不是其他同名作品。
- 已重新核对 Noble Master 官方 press kit `https://www.noblemaster.com/press/sheet.php?p=desert_stormfront`、官方 trailer `https://www.youtube.com/watch?v=7oZC6DpIUhQ`、官方海战图 `screen_dsf_v02_10.png` 与 `screen_dsf_v02_11.png`。官方素材以清晰舰种轮廓、紧贴单位的阵营 / 生命信息和短促直线弹道维持高密度战场可读性；本项目只借鉴这些原则，不复制素材或 UI。
- 当前 `addStructureBody(...)` 的 Coastal Battery 只有矩形 pad、椭圆 revetment / turret、单根炮管和阵营条；`showNavalGunSalvo(...)` 已把 Coastal Battery 与 Battleship 都表现为双发，模型与战斗反馈不一致。
- `fire(attacker:target:)` 已统一处理施工禁火、可见性、伤害、冷却、目标、伤害飘字和水面命中；本轮不得另建平行战斗系统。

## 目标

1. 在原 Coastal Battery gameplay footprint 和实体树内构建分层混凝土基座 / 炮床、护坡与沙袋、炮塔座圈、炮盾、两根平行长炮管、后膛、测距仪 / 阵营光学件、指挥掩体、弹药箱和通风细节；不改变碰撞、selection、health bar、施工或数值。
2. 让 Coastal Battery 的 `showNavalGunSalvo(...)` 炮口起点与新长炮管对齐，并通过 `showCoastalBatteryRecoil(...)` 增加双后坐轨迹、短促炮床冲击环与岸边尘浪；普通节点约 0.6 秒内移除。
3. 未知 Red Coastal Battery 不得通过炮线、炮口、后坐或尘浪泄露；未完工岸防炮仍不得开火。
4. 选择信息中的岸防职责文字从泛化 anti-ship 更新为可识别的 twin naval guns，同时保留 No sonar 和 Secure Coast 状态；真实攻击时继续优先显示目标 HP / 距离 / reload。
5. 新增 capture-only `coastal-battery` 场景：operational Blue Coastal Battery 在真实海岸 tile 锁定约 54% HP Red Battleship，冻结双发炮迹 / 弹体 / 炮口、后坐 / 尘浪、舰体命中 / 主副水柱，显示岸防炮射程圈、单选目标 HP / 距离 / reload、海岸地貌、小地图与单排命令条。
6. GitHub Actions 从十三次扩展为十四次独立启动，新增 `simulator-coastal-battery.png`。

## 非目标

- 不改变 Coastal Battery damage、range、cooldown、vision、cost、build time、placement、AI、target acquisition 或 `canAttack(...)`。
- 不增加 splash damage、击退、dot、弹药库存、升级或手动旋转炮塔。
- 不修改 Battleship / Carrier / Submarine / Shipyard / Sonar Buoy 模型。
- 不改变海军路径、潜艇隐身、声呐、任务、经济或胜负。
- 不新增外部依赖、图片素材或音频。
- 不运行本地 Xcode build、本地 simulator 或本地玩法探针。

## 当前架构依据

- `addStructureBody(for:to:)` 是 Coastal Battery 静态模型的唯一入口；模型节点随实体迷雾、施工、销毁和阵营状态统一管理。
- `fire(attacker:target:)` 是玩家与 AI 共用的直接火力结算入口；v4.93 已在炮位对玩家已知时调用 `showNavalGunSalvo(...)`，一次 attack cycle 的 `target.hp` 写入仍只有一次。
- `showNavalGunSalvo(...)` 已接收 `kind`、`faction` 和 capture-only `persistent`，可在 `kind == .coastalBattery` 时追加纯视觉 recoil helper，不改伤害链路。
- `prepareCICaptureScene()` 已按 `DESERT_CI_COMMAND_MARKER` 分流；新 marker 必须完全受 `DESERT_CI_CAPTURE_MODE=1` 限制。
- 海岸地形规则是 land tile 邻接 water；capture 可使用 row 17 / col 21 的真实海岸 tile，并把原 Shipyard 移到不重叠位置。
