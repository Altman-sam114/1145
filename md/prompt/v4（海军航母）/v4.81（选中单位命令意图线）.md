# v4.81 / 选中单位命令意图线

## 版本号与依据

v4.81。v4.80 三类命令落点瞬时反馈已验收；下一优先级是让玩家在选中单位后持续看清当前命令意图：移动目的地、攻击移动终点、直接攻击目标和 HOLD 站位。

## 视觉参考

- Noble Master Desert Stormfront 官方截图 / trailer：高对比短命令线、贴地低遮挡、按命令类型分色。
- 只借鉴“选中可读、分色、低遮挡”，不复制原素材。

## 目标

- 玩家选中且存活的己方机动单位显示命令意图线：
  - 直接攻击：红色线到 `attackTarget` 当前位置，仅当目标对玩家已知。
  - 攻击移动：琥珀色线到 `attackMoveDestination`。
  - 普通移动：青绿色线到最终 destination / path 终点。
  - 仅 HOLD：淡金色短线到 `holdPosition`。
- 优先级：attackTarget > attackMoveDestination > destination/path > holdPosition。
- 节点挂在实体 `commandIntentNode`，选中时刷新，取消选中隐藏；不改命令语义、寻路、战斗、AI、迷雾。
- CI air capture 场景中选中玩家空军已有 destination；额外给至少一架设置 attackTarget，使主截图可验证意图线；保留既有 command marker 探针。
- 同步 README、flow、flowchart、test.md。

## 非目标

- 不改触摸、编队、伤害、射程、AI、HUD 页映射。
- 不给敌方/AI 显示意图线。
- 不新增外部依赖或 XCTest。
- 不本地测试；不提交 `project.pbxproj` 团队号。

## 验收

- origin/main CI success，manifest SHA/version=v4.81。
- 截图真实战场，选中玩家单位可见分色意图线，不黑白屏。
- 八次（或既有）启动 PID 存活。
