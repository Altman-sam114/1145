# v4.75 / 空军投影与导弹尾迹

## 版本号与依据

v4.75。v4.74 已由最终 `origin/main` run `29123754769` artifact 验收通过；下一项从海战切换到空战辨识和命中反馈。

## 参考与背景

Agent X 核对了 Noble Master 官方 1 分钟 trailer、TouchGameplay 15 分钟 gameplay 和官方产品页。Desert Stormfront 的俯视空军包含 Helicopter、Plane，空中高度主要靠轮廓投影、移动偏移和高对比短弹道表达。本轮加入 HEL/JET 投影、Fighter/SAM/AA 导弹烟迹和空中命中环。

## 目标

- Helicopter 使用机身、尾梁、旋翼投影；Fighter 使用 jet 轮廓投影。
- 投影作为实体子节点遵守迷雾，移动时按方向调整偏移和透明度。
- Fighter、SAM Site、AA Truck 攻击改用烟迹、高亮轨迹和发光弹体。
- 导弹命中可见空军时显示双环和六向火花。
- CI 使用 `air` 焦点，稳定展示 Blue / Red HEL/JET、投影、导弹和命中环。

## 非目标

- 不修改飞行速度、高度、路径、攻击伤害、射速、射程、目标合法性、AI、Carrier guard、经济、任务、胜负或迷雾。
- 不新增粒子系统、外部素材、依赖或 XCTest target。
- 不运行本地测试、build、模拟器或探针。
- 不提交用户 `project.pbxproj` 团队号改动。

1. `GameEntity` 增加预创建 `airShadowNode`，配置 HEL/JET 轮廓并隐藏旧通用椭圆影子。
